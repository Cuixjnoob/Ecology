# 参数恢复分析报告

## 方法说明

- **参数化差异**：数据用 Ricker 形式 `x_{t+1} = x_t * exp(r + A·x)`，
  模型用 `x_{t+1} = x_t + lv_scale · x_t · tanh(r + A·x) + residual`。
  参数数值不可直接对比，因此使用**符号准确率 + 秩相关**作为核心指标。

## 初步结论

- 数值层面的参数恢复受参数化差异限制，不能期望 true ≈ pred
- 符号准确率和秩相关是更合理的评估方式
