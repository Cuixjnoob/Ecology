## 一、核心技术路线

### 1.1 研究问题

野外生态系统通常无法被完整观测。一个真实群落中可能包含数十个相互作用的物种，但持续监测能覆盖的往往只是其中一个子集。在这种条件下，研究者实际能拿到的是一个被截取的子系统：被观测物种的丰度时间序列 X ∈ R^{T×N}（N 个可观测物种、T 个时间步），而真正驱动这个子系统演化的若干未观测物种 h(t)，无法直接读出。

本研究关注如下问题：仅根据可观测物种的时间序列 X，能否反推出某个未观测物种 h(t) 的轨迹？该问题不是对未来进行预测，而是在已知观测窗口内对系统中"缺失"的那一部分进行重建。它对应若干生态学应用场景，例如识别食物网中尚未被直接采样的中介物种、估计实验设计中刻意排除物种的状态、以及从历史观测出发推断已经局部消失或采样不足的物种动态。

### 1.2 任务约束的合理性

本研究将任务严格限定为 n→1 设定，即一次只反推一个目标隐藏物种。这一约束并非任意为之。如果允许模型自由地输出多通道隐藏表示，模型容易学到一个数学上能够解释观测但不对应任何具体生态对象的低维潜变量。这种潜变量在重构指标上表现良好，但在生态学上无法解释。把目标限定为单一物种，可以强制学到的隐藏量必须与具体生态对象对齐，避免它退化为容纳任意残差的内部变量。

本研究同时设定了一条贯穿始终的约束：训练过程中绝对不使用 hidden 真值，也不使用任何由 hidden 真值派生出来的信号，包括监督目标、伪标签、由其计算得到的锚点、由其派生的初始化或先验。这一约束是出于研究目标的根本要求：方法最终的应用场景是真实野外数据，其中目标隐藏物种就是看不到、没有标签可用。任何形式的隐藏监督都会让方法在合成数据上"看起来好"，但在真实数据上根本无法运行。基于此，hidden 真值仅用于合成数据上的最终评估，不进入训练流程。

### 1.3 该问题不同于普通时间序列预测

普通的多变量时间序列预测，输入和输出在同一变量空间，模型只需学习一个映射 X[t+1] = f(X[t])，并以该映射在未来时间步的预测准确度为目标。本研究的目标变量 h(t) 在训练阶段从未出现，也没有任何标签可学。模型可依赖的唯一信号是：如果 h(t) 真的对可观测物种有动力学影响，则可观测物种的时间演化里必然留有 h(t) 的"足迹"，即用 X 自身无法解释的残差。整个方法的本质，是从可观测残差里把这种足迹反推出来。

这一反推面临两个根本困难。其一是可识别性：同一段可观测轨迹通常可以被多种 (h, dynamics) 组合所解释，从纯粹的数学角度并不存在唯一最优解。其二是信噪比：在真实数据上，hidden 对可观测物种的影响往往较弱，可观测残差里的能量主要来自采样噪声、未建模物种、外生环境扰动等，hidden 的真实贡献相对其他成分较小。本研究的方法学设计正是围绕这两个困难展开。

### 1.4 方法整体结构

本研究的最终方法由两部分组成：后验编码器（Posterior Encoder）从可观测序列输出 hidden 的后验分布；动力学模块（Dynamics）使用可观测变量与从后验中采样得到的 hidden 共同重构可观测变量自身的下一步演化。两者通过证据下界（ELBO）联合训练，并辅以多项结构性约束。

具体的输入输出关系如下：可观测序列 X 进入后验编码器，得到每个时间步 hidden 的高斯后验参数 q(h|X) = N(μ(X), σ²(X))；从该后验中抽取 hidden 样本，与可观测变量一同送入动力学模块，输出对可观测序列 log-ratio 的预测；预测与真实 log-ratio 之间的均方误差作为重构损失，与变分正则、多步前推一致性损失、反事实约束一并构成总损失。

### 1.5 后验编码器设计

后验编码器需要从可观测序列中提取足以约束 hidden 的信息。本研究采用了图神经网络（GNN）与 Takens 延迟嵌入相结合的设计。

GNN 用于建模物种间结构。在生态系统中，物种通过种间相互作用相关联，将节点设为物种、边设为相互作用，是一种与生态语义直接对应的建模方式。编码器内部的跨物种自注意力让任意时刻的所有物种相互"看到"，使得可观测物种的瞬时状态在物种间被充分融合。

Takens 延迟嵌入用于处理时间结构。任一时刻的物种丰度本身往往不足以确定系统当前所处的相空间位置，但将其与若干历史时刻 [x(t), x(t-1), x(t-2), x(t-4), x(t-8)] 拼接后，得到的局部相空间向量足以反映系统的轨迹形状。Takens 定理保证了在合适的延迟选择下，这种延迟堆叠能够恢复原系统的动力学几何。延迟嵌入与跨时间自注意力结合，使得编码器能从有限观测中重构系统的局部时间结构。

编码器的最终输出是每个时间步上 hidden 的后验参数 (μ(t), log σ(t))。变分形式使得 hidden 推断附带不确定性度量，也使得训练可以在标准的变分推断框架下进行。

### 1.6 动力学的残差分解

动力学模块的设计是本研究方法的核心环节。如果将动力学写成一个直接吃 (x, h) 输出 log-ratio 的统一函数，模型在训练中会出现严重的失败模式：当函数本身具有足够的表达力时，它可以仅靠可观测物种之间的相互作用解释可观测序列的演化，而完全绕开 hidden 输入，使 hidden 退化为对预测毫无影响的闲置变量。这一现象被项目早期的诊断实验明确观察到，是无监督设定下 hidden recovery 的典型陷阱。

为消除这一失败模式，本研究将动力学强制写成残差分解的形式：

```
log(x_{t+1} / x_t) = f_visible(x_t) + h_t · G(x_t)
```

其中 f_visible(x) 仅以可观测变量为输入，承担"可观测变量自身可解释部分"；G(x) 也仅以可观测变量为输入，输出每个物种对 hidden 的敏感度场；hidden 与 G 进行逐元素相乘后加入预测。这种结构具有一个硬约束：当 h = 0 时，hidden 分支的贡献严格为零。这使得"动力学绕开 hidden"在数学上变得不可能——任何对预测的有效贡献都必须经过 hidden 通道，模型再灵活也无法回避 hidden。

这一分解的另一层意义是语义上的："hidden 是可观测残差的最小必要解释器"，而不是一个能够吸收任意未解释项的内部变量。这使 hidden 的角色从"潜在因子"转化为"残差结构"，更接近其在生态学中的真实含义。

### 1.7 动力学消息层的形式选择

无论是 f_visible 还是 G，其内部都是物种级图神经网络。每条边 j → i 上的消息如何计算，是一个需要权衡的设计问题。

最直接的方案是软预设公式：在每条边上并行计算若干种生态学公式（例如线性项、Lotka-Volterra 双线性、Holling II 线性、Holling II 双线性），并使用一组可学习门控对这些公式做加权混合，使模型在训练中"自动选择"合适的形式。本研究的早期版本采用了这一方案。20 次随机种子的实验显示，所有数据集上各种公式的门控数值收敛到几乎相同的水平（约 0.12–0.18），并未出现"在 LV 数据上 LV 公式权重高、在 Holling 数据上 Holling 公式权重高"的预期分化。该方案在表达能力上等价于若干生态公式的等权线性混合，并不能真正"发现"具体的动力学形式。

本研究最终采用的方案是 MLP 主干配合 formula hints。每条边的消息由一个小型多层感知机计算，其输入除了物种状态值与可学习的物种嵌入向量外，还包括上述若干生态学公式的数值作为额外的输入特征：

```
m_{ij} = MLP([x_i, x_j, s_i, s_j, x_j, x_i·x_j, x_j/(1+α_j·x_j), x_i·x_j/(1+α_j·x_j)])
```

这里的关键在于：生态学公式作为多层感知机的输入提示，而非作为可强制选择的硬选项。多层感知机可以非线性地组合、加权、忽略这些公式，也可以在它们之外建立自己的依赖结构。这种设计具备两层意义：其一，公式提示提供了与生态学先验一致的归纳偏置，使训练较为稳定；其二，多层感知机的非线性组合能力覆盖了"公式线性混合"无法表达的形式，例如各公式之间的乘积、对数、指数等高阶组合。

这一选择是在"软预设"与"完全自由神经网络"之间寻得的中间路径，实验上证明优于两端。

### 1.8 多步前推一致性约束

仅以单步重构损失训练时，模型容易学到一类伪解：所学的"动力学函数"在每一个时间点上都能够正确预测下一步，但在多步前推时迅速发散。从形式上看这种伪解满足训练目标，但其学到的并不是真实系统的动力学函数，而是某种与真力学数值上局部对齐、结构上不一致的组合。

项目的诊断实验（详见仓库 `notes/2026-04-13_cvhi_residual_final.md`）发现，在某些早期版本上，将 hidden 真值代入训练好的动力学进行重构，重构误差反而高于使用编码器自身输出的 hidden。这一现象表明，模型学到的动力学并不真正接受真实 hidden 作为输入，而是构造了一个仅与编码器输出兼容的伪解。

为压缩这一伪解空间，本研究引入了 L1 多步前推一致性约束：从每个时间起点开始，以真值起步，让动力学模块自由前推若干步（默认 3 步），要求每一步的预测都与真实轨迹对齐。形式上：

```
x̂_0 = x_t
对于 k = 1, 2, 3：
    log_ratio_k = f_visible(x̂_{k-1}) + μ_{t+k-1} · G(x̂_{k-1})
    x̂_k = x̂_{k-1} · exp(clamp(log_ratio_k))

L_rollout = Σ_k w_k · MSE(log x̂_k, log x_{t+k})
```

其中 w_k 为递减权重，使较远步数的影响逐步减弱以保持训练稳定。该项损失迫使所学动力学具备多步自洽性：如果学到的是真力学，则多步预测自然成立；若是伪解，则在多步前推下必然漂移。该约束在项目实验中被证实能有效降低伪解的占比，是当前方法稳定性的重要来源。

### 1.9 反事实必要性约束

为进一步约束 hidden 的角色，本研究还设计了两项反事实损失。第一项要求"将 hidden 置零后的预测必须显著差于含 hidden 的预测"，即 hidden 不能成为对预测无影响的花瓶变量。第二项要求"将 hidden 沿时间轴打乱后的预测必须显著差于原始预测"，即 hidden 必须具有真实的时间结构，而不是逐时间步的独立噪声。两项反事实损失均以一个最小边际为下限，仅在边际不达标时给出梯度。

这两项约束与残差分解的硬约束互补：残差分解保证 hidden 通道存在，反事实损失保证 hidden 通道被实际使用且其内容具备时序意义。

### 1.10 方法路线的形成过程

当前的方法配置不是一次形成的。项目经历了若干次重要的方法学调整：早期版本曾尝试以 Linear Sparse + EM 的输出作为后验的锚点提供稳定性，但该锚点的计算过程间接使用了 hidden 真值，违反了项目的核心约束，因而被去除；其后曾尝试软预设公式选择（SoftForms），但实验表明门控未发生分化，该方案被替换为 MLP 主干配合 formula hints；曾尝试为 hidden 施加慢变量先验（频谱低通约束），但该假设在合成 LV 数据上被实验数据反驳——实测的高频能量与恢复质量呈正相关，先验假设不成立，相关项被移除。

每一次调整均由诊断实验给出明确依据。当前方法可视为在严格无监督约束下，针对若干已知失败模式逐一施加结构性约束之后所收敛的稳定配置。

## 二、模型设计与实验结果

### 2.1 模型结构概述

最终方法的整体结构为：

可观测序列 X ∈ R^{T×N} 输入后验编码器，编码器由若干 GNN 模块（含跨物种注意力与跨时间注意力）堆叠而成，输入特征通过 Takens 延迟嵌入扩展（lags = [1, 2, 4, 8] 或 [1, 2, 4, 8, 12]，依数据时间尺度而定）。编码器输出每个时间步上 hidden 的后验参数 (μ, log σ)。

动力学模块由两个并行的物种级图神经网络组成。f_visible 模块为两层结构，其内部每条边的消息由 MLP 配合若干生态公式提示项计算，邻居选择采用 top-k 注意力（k = 4）。G 模块为单层结构，输出每个物种的 hidden 敏感度。两个模块的输出按 `f_visible(x) + h · G(x)` 合成，得到对可观测序列 log-ratio 的最终预测。

每次前向计算时，从后验中抽取若干 hidden 样本（默认 2 个），分别参与重构。最终评估时，从后验中抽取 30 个样本并取均值作为 hidden 的点估计。

### 2.2 损失函数

总损失由若干部分构成：

```
L = L_recon                                       # 单步重构损失
  + β · KL[q(h|X) ‖ N(0, σ_prior²)]               # 变分正则
  + λ_necessary · ReLU(margin_null − Δ_null)       # 反事实必要性
  + λ_shuffle · ReLU(margin_shuf − Δ_shuf)         # 反事实时序结构
  + λ_energy · ReLU(min_energy − var(h))           # hidden 能量下限
  + λ_rollout · Σ_k w_k · MSE_k                    # 多步前推一致性
  + λ_smooth · ‖Δ²h‖²                             # 轻微平滑
  + λ_sparse · L1(动力学参数)                       # 参数稀疏
```

其中 σ_prior = 1，β = 0.03，rollout 步数为 3，权重为 (1.0, 0.5, 0.25)。训练采用三阶段调度：前 20% 仅训 f_visible，中间约 20% 线性提升 hidden 相关项的权重，其后阶段全部约束开启。

### 2.3 实验设置

本研究使用三个数据集进行验证。两个为合成数据，一个为真实野外观测数据。

合成 Lotka-Volterra 数据由仓库 `data/partial_lv_mvp.py` 生成，包含 5 个可观测物种与 1 个隐藏物种，时长 820 步，含环境驱动与稀疏脉冲扰动。系统遵循离散时间 Lotka-Volterra 形式，物种间含捕食—竞争混合相互作用。

合成 Holling 数据由仓库 `data/partial_nonlinear_mvp.py` 生成，规模与 LV 数据相同，但动力学包含 Holling II 型饱和、Allee 效应与时滞耦合，整体非线性程度显著高于 LV。

真实数据采用 Portal Project 的长期啮齿动物监测数据。该项目位于美国 Arizona 州 Chihuahuan 沙漠，自 1977 年至今持续运行，本研究所用文件覆盖至 2020 年之后，共 520 个月度观测、41 个物种。预处理时取累计捕获量覆盖全样本 95.47% 的前 12 个物种作为近似完整群落，其余稀有物种作为未建模残差处理。固定 OT（Onychomys torridus，南方蚱蜢鼠）作为目标隐藏物种，其余 11 个物种作为可观测变量。该选择是出于 OT 在该群落中具有中等耦合强度，便于检验方法在中等可识别性条件下的表现。

正式统计实验采用 20 个独立随机种子的协议，对应仓库中 `scripts/cvhi_residual_20seeds_formal.py`。每个种子训练 300 个 epoch，最终模型由验证段重构损失最低的检查点选定。评估指标以 scale-invariant Pearson 相关为主，即将模型输出经线性回归对齐到真值后计算相关系数。

这一现象的合理解释如下：监督基线 Linear Sparse + EM 通过线性投影从可观测残差中提取与 hidden 真值线性相关的成分，其性能上限受限于线性投影本身的表达能力。在含有 Holling II 饱和、Allee 效应与时滞耦合的非线性系统上，hidden 在可观测残差中的真实印迹具有非线性结构，监督的线性方法只能捕获其中线性可分的一部分。本方法的多层感知机消息层具备非线性组合能力，可以利用残差中线性方法无法触及的信息，因而尽管不使用任何监督，仍能在该数据集上得到比监督方法更高的相关系数。

Holling 数据上 5 个种子落在 0.30–0.37 之间，与 LV 数据的失败模式类似，仍属于少数被伪解所吸引的情形。该现象进一步反映了无监督设定下可识别性问题对方法稳定性的影响。

Portal 数据上最佳种子的 Pearson 0.307 对应监督基线的 87%，均值 0.140 对应监督基线的 40%。该结果需要分两个层次理解。

第一是结构层。将 hidden 真值直接代入训练好的动力学进行重构，与使用编码器自身输出的 hidden 进行重构相比，重构误差几乎相同（d_ratio = 1.055）。这一指标提供了较强的结构性证据：模型学到的动力学在结构上与真实 hidden 兼容，不是仅与编码器自身输出兼容的伪解。在合成数据上（即使是当前定型方法），d_ratio 通常处于 4–13 之间，远高于 Portal 数据上的 1.06。Portal 上更接近理想值 1 的 d_ratio，反映了真实数据的较低自由度对模型构成了反向约束。

第二是数据层。监督基线 0.353 本质上接近"可观测残差中能与真实 hidden 建立线性对齐的成分上限"。任何无监督方法都受到同样的物理上限约束，达到该上限的 87% 已经接近此数据集的可识别性边界。Portal OT 是食虫的 Onychomys 属物种，与多数可观测物种（食种子的 granivore）在生态位上分离，相互作用强度本来就较弱。这一约束来自真实生态系统的结构，而非方法学缺陷。

### 2.7 主要消融观察

仓库中的 `scripts/cvhi_residual_backbone_compare.py` 直接对比了软预设公式方案（SoftForms）与 MLP 配合 formula hints 方案在 LV 与 Holling 上的表现（5 个种子，其他配置一致）：

| 主干形式 | LV 均值 | LV 最大 | Holling 均值 | Holling 最大 |
|---|---|---|---|---|
| SoftForms（软预设公式） | 0.752 | 0.906 | 0.224 | 0.922 |
| MLP + formula hints | 0.821 | 0.874 | 0.679 | 0.863 |

软预设方案在 Holling 上的均值仅为 0.224，5 个种子中 4 个落入 0.00–0.12 区间，仅 1 个种子达到接近完美的 0.92。MLP 配合 formula hints 方案将 Holling 上的均值提升至 0.679，5 个种子全部稳定在 0.30 以上、其中 4 个超过 0.65。这一对比说明：在 Holling 这类含强非线性的系统上，单纯的"软选择若干预设公式"不足以提供必要的表达能力，而将公式作为高表达力主干的输入提示，能在保留生态学先验的同时提供足够的非线性组合能力。

关于多步前推一致性约束的有效性，对比早期未引入该项的版本，Portal 数据上的 d_ratio 从 1.10–3.01 范围下降至 1.03–1.06 范围，提示模型学到的动力学与真实系统的兼容性显著提升。该项约束被保留为最终方法的重要组成部分。

关于 hidden 慢变量先验，合成 LV 数据上测得 ρ(高频能量比例, Pearson) = +0.79，表明 hidden 中高频能量的比例与恢复质量呈正相关。这意味着合成 hidden 本身就含有不可忽视的高频成分，强制低频会反向损害恢复质量。该先验项在最终方法中被移除。

### 2.8 当前方法相较早期版本的稳定性

5 个种子与 20 个种子的对比反映了样本规模对方差估计的影响：

| 数据集 | 5 种子均值 | 20 种子均值 | 5 种子标准差 | 20 种子标准差 |
|---|---|---|---|---|
| LV | 0.82 | 0.73 | 0.05 | 0.22 |
| Holling | 0.68 | 0.74 | 0.20 | 0.19 |
| Portal | 0.17 | 0.14 | 0.09 | 0.09 |

20 个种子的统计揭示出 LV 与 Holling 上的真实分布是双峰的：约四分之三的种子收敛到合理解，其余四分之一被伪解吸引。这一现象在 5 个种子的小样本下被掩盖。Portal 上的方差则较为稳定，反映了真实数据较低的自由度对模型的额外约束。

### 3.2 阶段性核心结果

本阶段最具分量的两项结果如下。

第一，在合成 Holling 数据上，无监督方法的均值 Pearson（0.738）超过了使用 hidden 监督投影的 Linear Sparse + EM（0.620）。这一结果说明，在含有典型生态饱和非线性的系统上，纯无监督方法在适当架构设计下不仅可行，而且可能优于受限于线性形式的监督方法。该现象在生态时间序列文献中相对少见。

第二，在 Portal 真实数据上得到的 d_ratio = 1.055，提供了较强的结构性证据：模型学到的动力学与真实 hidden 兼容，并非数值上偶然成立的伪解。这一指标的设计本身是从早期诊断实验中发展出的，反映了项目方法学体系的成熟度。

### 3.3 主要遗留问题：可识别性的天然困境

本研究的核心难点是结构性的：在严格无 hidden 监督的设定下，部分观测条件下的 n→1 hidden recovery 的目标函数本身没有唯一解。

形式上，给定可观测轨迹 X，存在多组 (h, dynamics) 满足相同的观测重构。本研究通过残差分解、反事实约束、多步前推三个结构性正则收缩了解空间，但并未将其完全收敛到唯一点。20 个种子实验中观察到的方差（LV 标准差 0.22，Holling 标准差 0.19）即是这种"缩小但仍存在多解"的直接表现。少数种子落入伪解局部极小（LV 与 Holling 各自约 5/20），这并非优化失败，而是目标函数在该区域同样具有较低的损失值。

这一现象同时影响了无监督的种子选择策略。在 LV 数据上观测到 ρ(验证重构, Pearson) = +0.728，即验证重构损失最低的种子反而对应较差的 Pearson。这是因为最低验证重构的种子中，既包含真正的良好解，也包含过度拟合可观测轨迹的伪解，二者难以靠重构指标本身区分。这一观察意味着当前架构下，无监督的"按验证损失选种"集成策略并不可靠。

### 3.4 真实数据上的恢复对象

合成数据上，"hidden 真值"是数据生成过程明确写定的某个变量，恢复目标是清晰的。真实数据上的情况更复杂。

Portal 数据中 OT 的月度波动同时受到多个因素驱动：自身种群动力学（被本研究当作 hidden 的部分）、降雨与季节等外生气候驱动（未在可观测变量中编码）、月度采样过程中的统计噪声、以及与未观测的稀有物种（top-12 之外的 29 个物种）的相互作用。当模型从可观测残差中反推一个 hidden 时，它能获得的最多是 OT 真值与上述各项干扰的混合投影。Pearson 最大 0.31 反映的并不是"模型拟合不到位"，而是"可观测残差中能与 OT 真值线性对齐的成分本身有限"。

监督基线 0.353 在数学上接近这一上限的上界。无监督方法接近其 87% 已触及该数据集的可识别性边界。这一结论一方面证明方法在真实数据上确实在工作，另一方面也说明，对 Portal 这类生态数据而言，所有无监督 hidden recovery 方法都受到一个由数据本身决定的硬天花板的约束。

### 3.5 当前阶段的整体定位

综合以上结果，本项目当前阶段的整体定位可以表述为：方法的架构与训练范式已经定型，但距离"对真实生态系统的隐藏物种进行完全恢复"仍有距离。

定型部分包括：模型架构、损失函数、训练调度、诊断指标体系与统计协议均已固定，进一步的局部调参很难再带来质变。

未完成部分主要在三个方向：其一，是否存在一种在不依赖 hidden 真值的前提下自动剔除伪解种子的机制；其二，能否在更多类型的真实生态数据上得到与 Portal 同等量级的结构性证据；其三，能否在保持当前方法核心的前提下对 n→1 设定做有限的推广。

## 四、后期规划

### 4.1 近期工作

近期工作以中期报告之后的若干周为时间窗口，聚焦于将当前的方法定型转化为可投稿状态。

第一，完成完整的逐组件消融实验。当前已有的对比集中在主干形式的选择（SoftForms 对比 MLP + formula hints），尚需对其他关键组件做相同规模（20 种子）的独立消融，包括：是否引入 formula hints、是否引入多步前推一致性、是否使用残差分解、是否引入两项反事实约束。这一组消融是论文方法部分必要的支撑。

第二，进行跨 hidden 物种的稳健性检验。当前 Portal 实验仅以 OT 作为隐藏物种。需要补充以 DO（Dipodomys ordii，Ord 氏跳鼠）、PP（Chaetodipus penicillatus，沙漠口袋鼠）、PB 等若干物种作为隐藏目标，验证方法不是仅对某一个特定物种生效。

第三，探索更可靠的无监督选种机制。当前验证重构损失在 LV 与 Holling 上呈反向相关，无法直接用作种子过滤器。可尝试的方向包括：使用更长 rollout（步数 5–10）的发散程度作为额外指标；使用 d_ratio 之外的若干无监督诊断指标的组合作为综合选种依据。

第四，开展论文初稿撰写。当前已积累的实验结果与方法学讨论已经足以支撑一份完整的论文，包含方法（四个核心组件加诊断协议）、合成实验（LV 与 Holling）、真实实验（Portal）、消融与不足讨论。先完成初稿，再依据初稿审视的感受决定后续实验补充。

### 4.2 进一步工作

进一步工作在数月时间窗口内推进，目标在于扩展方法的适用范围与提升论文层级。

第一，更广的真实数据覆盖。Portal 是单一沙漠群落数据。补充一到两个不同类型的真实生态时间序列（例如 Lake Mendota 长期浮游生物数据、BCI 森林监测数据等），可以将"在 Portal 上工作"提升为"在多类真实生态系统上工作"，论文外推性可获得显著增强。

第二，外生驱动的显式建模。当前架构无法区分"hidden 物种贡献"与"环境驱动贡献"，二者会同时进入残差。后期可在动力学中显式增加外生驱动通道，使用类似的残差分解结构将环境分量分离出来。这一方向需要相应的外生协变量数据，因此放在进一步工作中。

第三，对 n→1 设定的有限推广。严格 n→1 是当前方案的核心约束，但部分场景需要同时恢复多个隐藏物种。难点在于多通道 hidden 容易退化为不可解释的内部潜变量（这正是当初设定 n→1 的原因）。如果能找到约束多通道 hidden 各自对应具体物种的机制（例如对每个通道施加不同的结构先验），可适度扩展任务范围。

第四，可解释性的深化。当前 MLP 内部对各 formula hint 的实际依赖程度无法直接读出。可通过扰动敏感度分析（例如关闭某个 hint 重训并观察性能下降幅度）获得"模型在该数据上实际依赖哪些生态公式"的可解释性结果，以增强论文的方法可解释性叙事。

第五，可识别性的形式化讨论。在论文方法部分加入一段对可识别性问题的形式化讨论：在何种条件下 n→1 部分观测下的 hidden recovery 在无监督设定下具有唯一解；本研究的若干结构约束（残差分解、反事实约束、多步前推）在可识别性意义上各自扮演什么角色。这部分若能讲清楚，对论文的层级有直接帮助。

## 当前工作的阶段性水平与潜在投稿层级分析

对当前项目进展进行综合评估，可以从若干维度交叉判断。

从方法创新性角度看，本研究的核心方法 CVHI_Residual 包含若干独立设计：将动力学强制写为 `f_visible(x) + h · G(x)` 的残差分解形式，是为消除"hidden 被架空"失败模式而专门构造的硬约束；两项反事实损失（hidden 置零与 hidden 时序打乱）是为约束 hidden 的必要性与时序结构而专门构造；MLP 主干配合 formula hints 是介于"软预设"与"完全自由"之间的中间路径，在文献中没有完全相同的工作。多步前推一致性约束本身在时序学习领域并非新物，但将其作为"伪解过滤器"用于 hidden recovery 是项目特有的角度。整体方法具有独立创新性，但单点上未见颠覆性突破。

从实验完整性角度看，已完成 20 种子的正式统计、3 个数据集（2 个合成 + 1 个真实）、配套的诊断指标（d_ratio、ρ(val, P) 等）。但完整的逐组件消融实验目前仅覆盖部分组件，论文评审通常会要求补齐。

从真实数据表现角度看，Portal 上最大 Pearson 0.307、均值 0.140，相对监督基线最大占 87%、均值占 40%。这一数字若定位为"在严格无监督设定下逼近监督基线 87%"是合理叙事；若定位为"全面超越现有方法"则不成立。Portal 上的 d_ratio = 1.06 是该叙事的核心结构性支撑。

从论文叙事完整性角度看，方法的演化路径有完整记录，诊断驱动的设计过程（即从已观察到的失败模式出发、有针对性地施加结构约束）是论文方法部分的独特优势。多数同类工作难以清楚说明"为何最终选择此种方法"，本研究在此方面具有完整的失败—诊断—修复链条可供论述。研究的限制部分（可识别性的固有困难、按验证损失选种的失效、Portal 的物理上限）在仓库中也已有诚实的讨论。

从稳定性角度看，20 种子中 LV 与 Holling 各自约 5/20 落入伪解局部极小，Portal 上的方差为 0.087。这并非"高度稳定"的方法，但与早期 SoftForms 阶段 4/5 失败的稳定性相比已有质的改善。

综合上述维度，当前工作的阶段性水平最接近 SCI 三区中上至二区下沿候选区间。具体而言，以现有结果直接整理投稿，落点在强三区或二区下沿是合理的预期。是否能站稳二区下沿，主要取决于以下三点：能否在中期报告之后完成完整的 20 种子级逐组件消融；能否在论文方法部分将残差分解、反事实约束、多步前推三项各自的设计动机讲到位（不仅说明"做了什么"，更说明"诊断了什么问题、为何这样修"）；以及能否将 Holling 上无监督超越监督基线的现象上升为论文 discussion 中的方法学论断。

进一步上升至中等二区或更高水平的关键限制因素有两个。其一是真实数据的覆盖广度。仅有 Portal 一个真实数据集，难以支撑"方法对生态系统普遍有效"的强论断。若能再增加一到两个不同类型的真实生态时间序列数据并取得相似量级的结果，论文层级可获得实质性提升。其二是可识别性问题的处理深度。当前结果通过结构约束"绕开"了可识别性，但并未"解决"它，仍有约四分之一的种子落入伪解。若能给出某种形式化的统计叙述（例如在某种条件下解集是唯一的），或一个能完全自动剔除伪解的无监督机制，论文层级可再上一级。

如果当前阶段直接整理投稿，强三区基本稳健，冲二区有概率但需补完整消融。若再投入数周时间完成完整消融与一到两个额外真实数据集的验证，二区中下游的命中概率会明显提高。继续向上则需在理论侧或在更广泛真实数据上取得质的突破。
