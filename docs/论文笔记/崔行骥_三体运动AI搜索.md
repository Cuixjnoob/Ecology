## 研究问题

在不预设稳定构型的情况下，如何利用数据驱动的 AI 搜索方法在平面三体系统的初始条件空间中自动发现长寿命稳定轨道？

## 核心方法

### 问题建模
- 平面三体系统，质量归一化 $\sum m_i = 1$，$G = 1$
- 质心固定于原点，零总动量约束
- 有效参数空间维度：$d = 15 - 5 = 10$
- 评价指标：存活时间 $T_{\text{survive}}$（首次碰撞或逃逸的时刻）
- 数值积分：RK4/RK45

### 三阶段搜索策略

**阶段 1 - 精英随机搜索（ERS）**:
- 随机采样 → 按存活时间排序 → 取前 K 个精英
- 对精英加高斯噪声扰动：$\theta_{\text{new}} = \theta_{\text{elite}} + \sigma_g \xi$
- 指数退火策略：$\sigma_g = \sigma_0 \alpha^g$

**阶段 2 - 交叉熵方法（CEM）**:
- 用精英样本的统计量更新高斯采样分布
- $\mu_{g+1} = \frac{1}{K}\sum_{\theta \in E^{(g)}} \theta$，$\Sigma_{g+1}$ 类似更新
- 引入多高斯混合分布 + K-means 聚类捕获多稳态解

**阶段 3 - Softmax 加权 + PINN 辅助**:
- 族内基于得分的 softmax 权重更新分布参数
- 简易 PINN（MLP 二分类器）快速筛选潜在优良样本
- 门控策略：高置信度时用 PINN，否则回退真实积分

### 截断与删失处理
- 尾部截断机制提升 GPU 吞吐率
- 右删失样本引入保守折扣因子 $\alpha$

## 关键发现与结论

1. **自发涌现的结构**: 从随机初始条件出发，算法自动发现了长寿命轨道，后期趋于收敛到环带型（双星螺旋）结构
2. **结构偏置**: 仅通过软性探索激励不足以克服存活时间目标导致的结构偏置
3. **环带结构的稳定性优势**: 该结构更容易保持有界、避免近距碰撞、减少能量发散
4. **计算资源限制**: 两天 66 代迭代下获得的最佳结果展示了混沌系统中的典型稳定构型

## 重要公式/概念

| 概念 | 含义 |
|------|------|
| **精英随机搜索** | 保留高分样本并在其附近扰动搜索 |
| **交叉熵方法** | 通过极大似然更新采样分布参数 |
| **多高斯混合模型** | 捕获参数空间中的多稳态解家族 |
| **PINN 门控** | 神经网络预判 + 不确定性判据的混合决策 |

## 与我们项目的关联

**中等关联**。

1. **混沌动力系统搜索**: 三体问题是经典混沌系统，其搜索方法思路（精英搜索 + 交叉熵 + 神经网络代理模型）可迁移到我们在混沌生态系统参数空间中的搜索问题。
2. **PINN 代理模型**: 用简化神经网络快速筛选候选方案的思路，可用于加速我们的 GNN 模型超参数搜索。
3. **混沌系统中的稳定结构**: 文章展示了混沌系统中的长寿命稳定结构如何自动涌现，这与我们研究混沌生态系统中的隐含规律有共鸣。
4. **与 Takens 嵌入的联系**: 三体系统是典型的低维混沌系统，其轨道重构与 Takens 延迟嵌入理论密切相关。

## 一句话总结

利用精英随机搜索、交叉熵方法和 PINN 辅助筛选的三阶段 AI 搜索策略，在平面三体系统中自动发现长寿命稳定轨道，展示了数据驱动方法在复杂动力系统结构探索中的潜力。
