# [Ovaskainen et al. 2016] Using Latent Variable Models to Identify Large Networks of Species-to-Species Associations at Different Spatial Scales

## 基本信息
- **作者**: Otso Ovaskainen, Nerea Abrego, Panu Halme, David Dunson
- **期刊**: Methods in Ecology and Evolution, 7: 549-555
- **年份**: 2016（发表），2015（接收）
- **类型**: 研究论文

## 研究问题
如何从物种出现/丰度数据和环境协变量中推断大型物种间关联网络？如何在多个空间尺度上同时分析物种共现模式，并区分环境过滤和生物交互的贡献？

## 核心方法
### 分层联合物种分布模型
- **数据结构**: 98种真菌，采样于29片山毛榉林中的230个样方，约22,500个枯木单元
- **模型框架**:
  - Probit回归: y_{ij} = 1 if z_{ij} > 0
  - 潜在出现分数: z_{ij} = L_{ij} + ε^R_{ij} + ε^P_{P(i)j} + ε^F_{F(i)j} + ε_{ij}
  - L_{ij}: 环境协变量的固定效应
  - 三层随机效应: 资源单元(R)、样方(P)、森林(F)

### 潜变量方法
- 在每个空间尺度上用潜变量参数化方差-协方差矩阵: ε^R_{ij} = Σ_k η^R_{ik} λ^R_{kj}
- 潜变量可代表缺失环境协变量或生态交互的结果
- 使用稀疏无限因子先验（Bhattacharya & Dunson 2011）自动确定潜因子数量

### 关键扩展
1. **多尺度潜变量**: 同时在资源单元、样方、森林三个空间尺度估计关联
2. **条件预测**: 利用其他物种的出现情况预测目标物种
3. **偏相关**: 除相关矩阵外还估计偏相关矩阵，后者更可能反映因果关系
4. **贝叶斯推断**: 使用Gibbs采样

## 关键发现与结论
1. **物种共现信息翻倍预测能力**: 考虑其他物种出现情况后，模型预测力（Tjur R²）大约翻倍
2. **不同尺度网络不同**: 相关性和偏相关性在不同空间尺度上呈现不同的网络结构
3. **自然林和管理林网络一致**: 物种关联在两种林型中保持一致
4. **方差分解**: 随机效应（物种关联）解释了大量变异，固定效应（环境）解释有限
5. 潜变量方法使大规模物种关联矩阵的估计变得可行（98个物种）

## 与我们项目的关联
- **JSDM是核心baseline**: 联合物种分布模型是推断物种间关联的经典方法，可直接与我们的GNN方法对比
- **潜变量 ↔ VAE**: Ovaskainen的潜变量建模思路与我们VAE中的潜空间概念一致——都是用低维潜变量解释高维物种共现模式
- **多尺度结构**: 生态系统中物种交互在不同尺度表现不同，提示我们的GNN需要考虑多尺度
- **条件预测能力**: 论文证明物种间信息可显著提升预测，支持我们利用已知物种恢复未知物种的方法论
- **关键局限**: JSDM是静态快照模型，不建模时间动态，这是我们GNN+VAE方法的核心优势

## 一句话总结
Ovaskainen 2016提出了基于潜变量的多尺度联合物种分布模型，证明物种间关联信息可显著提升预测能力，其潜变量建模思路与我们VAE方法高度相关，但缺乏时间动态建模是其主要局限。
