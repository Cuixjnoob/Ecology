## 研究问题

在多尺度系统模拟中，闭合问题（closure problem）无处不在——某些量或过程无法被完全描述，但又影响模拟精度。如何将传统物理建模与数据驱动的机器学习方法结合，用科学机器学习（SciML）解决闭合问题？

## 核心方法

### 闭合模型的分类体系

按照物理知识的融入程度，论文将闭合模型分为三类：

**1. 白盒模型（Physics-based）**:
- 完全基于物理方程
- 例如：LES 中的 Smagorinsky 模型

**2. 灰盒模型（Hybrid）**:
- 在物理方程中嵌入神经网络作为未知项
- **残差学习（Residual Learning）**: $\frac{du}{dt} = f_{\text{physics}}(u) + \mathcal{N\!N}(u)$
- **参数学习**: 用神经网络预测物理模型中的参数

**3. 黑盒模型（Data-driven）**:
- 纯数据驱动，无显式物理方程
- 例如：直接用神经网络学习动力学

### 学习目标的分类

**先验学习（A Priori Learning）**:
- 从 DNS/高保真数据中直接学习闭合项
- 损失函数：$\mathcal{L} = \|c_{\text{predicted}} - c_{\text{true}}\|^2$
- 优点：训练简单；缺点：部署后可能误差累积

**后验学习（A Posteriori Learning）**:
- 将闭合模型嵌入求解器，通过整体轨迹误差来训练
- 需要通过 ODE/PDE 求解器反向传播梯度
- 优点：部署稳定；缺点：训练复杂且计算昂贵

### 物理约束的强制方式

**软约束**: 通过损失函数惩罚项（如守恒律违反的 $L^2$ 范数）

**硬约束**: 通过架构设计保证约束自动满足
- 例如：质量守恒 → 学习流函数 $\psi$，速度 $u = \nabla \times \psi$ 自动满足 $\nabla \cdot u = 0$
- 等变神经网络保证对称性

### 与其他领域的联系
- **Mori-Zwanzig 理论**: 提供了精确的闭合形式（记忆项 + 噪声项），但通常不可计算
- **逆问题**: 闭合模型的训练本质上是一个逆问题
- **多保真方法**: 利用粗模型和细模型之间的关联

## 关键发现与结论

1. **先验 vs 后验的差距**: 先验训练效果好不代表后验部署效果好，误差累积是主要问题
2. **物理约束至关重要**: 无约束的神经网络容易违反守恒律、对称性等，导致非物理行为
3. **离散化不变性**: 理想的闭合模型应该与空间/时间离散化方案无关，但当前方法大多做不到
4. **可解释性与泛化性**: 学到的闭合模型是否可泛化到训练之外的条件，以及是否可解释，仍是重大挑战
5. **推荐的实践**: 灰盒模型（物理 + ML）通常优于纯黑盒或纯白盒

## 重要公式/概念

| 概念 | 含义 |
|------|------|
| **闭合问题** | 简化模型中未知项的建模问题 |
| **残差学习** | 用神经网络学习物理模型的残差/修正项 |
| **先验/后验学习** | 离线匹配闭合项 vs 在线匹配轨迹 |
| **Mori-Zwanzig** | 将完整动力学精确分解为已知项 + 记忆项 + 噪声项 |
| **等变网络** | 保持物理对称性（旋转、平移等）的神经网络架构 |
| **Neural ODE** | 将神经网络嵌入 ODE 求解器，支持连续时间建模 |

## 与我们项目的关联

**高度关联**。

1. **核心方法论指导**: 我们的 CVHI/GNN 方法本质上是一个闭合模型——用神经网络补充隐藏物种对可观测物种的影响（即"闭合"部分观测系统的动力学）。

2. **残差学习框架**: 论文中 $\frac{du}{dt} = f_{\text{known}}(u) + \mathcal{NN}(u)$ 的灰盒框架与我们的方法直接对应：$f_{\text{known}}$ 是可观测物种之间的已知交互，$\mathcal{NN}$ 学习隐藏物种的影响。

3. **先验 vs 后验训练**: 我们需要认真考虑训练策略。先验训练（直接匹配隐藏影响项）简单但可能不稳定；后验训练（通过 ODE 求解器反传）更稳定但计算更贵。

4. **物理约束**: 生态系统有自然约束（种群非负、质量守恒等），论文提供了将这些约束融入 ML 模型的系统方法论。

5. **Mori-Zwanzig 理论**: 为我们提供了理论框架——隐藏物种的影响可以分解为马尔可夫项（瞬时影响）和记忆项（历史依赖影响），这与延迟嵌入思想一致。
