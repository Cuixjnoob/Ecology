- **年份**: 2024
- **期刊**: Methods in Ecology and Evolution, 15: 1247-1260
- **DOI**: 10.1111/2041-210X.14358

## 研究问题

如何在缺乏性状（trait）信息的情况下，仅利用已知的交互矩阵，预测生态网络中未知的物种间交互关系？特别是，现有方法普遍缺乏对模型预测的交互的验证。

## 核心方法

### 自组织映射（SOM）预测框架

1. **输入**: 物种交互矩阵（已知的部分交互关系）
2. **SOM 训练**: 使用自组织映射（一种无监督机器学习算法）对交互矩阵进行学习，将节点按交互模式分组到输出层
3. **交互预测**: 训练后的模型为每对物种输出交互得分（interaction score）
4. **阈值确定**: 使用 F1 分数最大化策略，设定阈值将连续得分转换为二元交互/非交互判断
5. **验证**: 使用分类学验证（taxonomic validation）和文献验证两种独立方法

### 应用的数据集
- 三个单部（unipartite）元网络：KF（淡水鱼）、TEU（欧洲四足动物）、BS（巴伦支海）
- 一个双部（bipartite）元网络：BAP（巴西蚁-植物）

## 关键发现与结论

1. **高预测性能**: F1 分数达 0.84-0.98，ROC 达 0.88-0.99，大型网络表现优于小型网络
2. **误预测率极低**: 假阳性率仅 0.13%-0.72%，表明模型很少预测不存在的交互
3. **无需性状数据**: 纯粹基于交互矩阵的拓扑结构进行预测，适用于性状数据缺失的物种
4. **分类学一致性**: 预测的新交互与已知分类学关系一致（如同属/科物种间的交互模式）
5. **文献验证**: 约 45-73% 的新预测交互可在已有文献中找到支持证据

## 重要公式/概念

| 概念 | 含义 |
|------|------|
| **SOM（自组织映射）** | Kohonen 无监督神经网络，将高维数据映射到二维拓扑空间 |
| **元网络（Metaweb）** | 区域尺度上所有潜在物种交互的汇总网络 |
| **F1 Score 最大化** | 选择使 precision-recall 平衡最优的阈值 |
| **TSS（True Skill Statistic）** | 敏感度+特异度-1，排除随机预测的效果 |

## 与我们项目的关联

**中等关联**。

1. **交互网络补全**: 我们的 hidden recovery 任务本质上是"部分观测网络的补全"——从可观测物种的动态中推断隐藏物种的存在和交互。Adhurya 的方法展示了从不完整交互矩阵中预测缺失交互的可行性。

2. **无监督方法的启发**: SOM 作为无监督方法不需要标签数据，这与我们希望在缺乏隐藏物种"真值"的情况下进行推断的设定相似。

3. **局限性**: 该方法是静态的（只考虑交互矩阵的拓扑结构），不考虑时间动态。我们的项目需要从时间序列动态中恢复隐藏物种，因此需要动态建模能力。

4. **验证思路**: 使用分类学一致性和文献证据来验证预测的交互，为我们验证 hidden recovery 结果提供了参考思路。

## 一句话总结

利用自组织映射从已知交互矩阵中无监督地预测生态网络中缺失的物种交互，在多种元网络上取得了高精度且假阳性极低的预测效果。
