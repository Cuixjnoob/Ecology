# [Karl et al. 2017] Deep Variational Bayes Filters: Unsupervised Learning of State Space Models from Raw Data

## 基本信息
- **作者**: Maximilian Karl, Maximilian Soelch, Justin Bayer, Patrick van der Smagt
- **机构**: Data Lab, Volkswagen Group, Munchen, Germany
- **年份**: 2016 (arXiv: 1605.06432)
- **发表**: ICLR 2017
- **领域**: 状态空间模型、变分推断、动态系统辨识

## 研究问题
如何从原始高维观测数据（如图像序列）中以无监督方式学习和辨识潜在马尔可夫状态空间模型？核心挑战有两点：(1) 辨识——从数据中学习系统动力学；(2) 推断——从观测中推断潜在系统变量。两者相互竞争：更强大的系统表示需要更复杂的推断，而高效的推断（如卡尔曼滤波）则限制了系统的复杂度。

已有方法（如STORN、VRNN、DKF）虽然在压缩（重构）方面表现良好，但潜空间未能包含完整信息（例如缺少速度信息），导致无法进行可靠的长期生成预测。

## 核心方法/架构

### 关键创新：重参数化状态转移
DVBF的核心贡献在于**强制潜空间服从状态转移**，而非相反。通过将状态转移函数重参数化为确定性函数：

$$z_{t+1} = f(z_t, u_t, \beta_t)$$

其中 beta_t 为随机转移参数（由识别模型推断），而非直接推断潜状态 z_t。这使得误差可以通过时间反向传播，迫使潜空间编码对预测有用的完整信息。

### 状态空间模型假设
- **发射模型**（观测独立性）：p(x_{1:T}|z_{1:T}) = prod_t p(x_t|z_t)
- **转移模型**（马尔可夫性）：p(z_{1:T}|beta_{1:T}, u_{1:T}) = prod_t p(z_{t+1}|z_t, u_t, beta_t)
- 当前状态 z_t 包含关于当前观测和下一状态的全部必要信息

### 转移参数分解
将 beta_t 分解为：
- **w_t**：样本相关的过程噪声，可从输入数据推断（类似卡尔曼滤波中的校正项）
- **v_t**：通用的转移参数，样本无关（仅在训练时从数据推断）

识别模型的自然分解：q_phi(beta_{1:T}|x_{1:T}) = q_phi(w_{1:T}|x_{1:T}) * q_phi(v_{1:T})

### DVBF下界
$$\mathcal{L}_{DVBF} = \mathbb{E}_{q_\phi}[\ln p_\theta(x_{1:T}|z_{1:T})] - \text{KL}(q_\phi(\beta_{1:T}|x_{1:T}) \| p(\beta_{1:T}))$$

使用退火（annealing）版本改善优化：c_i = max(1, 0.01 + i/T_A)

### 局部线性转移（DVBF-LL）
具体的转移参数化实例：
$$z_{t+1} = A_t z_t + B_t u_t + C_t w_t$$

其中 A_t, B_t, C_t 为 M 个全局线性系统的状态依赖加权组合：
- alpha_t = f_psi(z_t, u_t) -- 混合权重
- A_t = sum_i alpha_i^(i) A^(i), 类似于局部线性化

## 关键发现与结论
1. **完整信息潜空间**：DVBF成功从摆钟图像序列中恢复了角度和角速度（R^2=0.916），而DKF几乎无法提取速度信息（R^2=0.035）
2. **稳定长期预测**：DVBF能进行远超训练序列长度的可靠生成采样
3. **弹跳球实验**：DVBF从256像素中正确提取了2D位置和速度的4维状态空间
4. **状态空间假设的强制执行**：通过重参数化转移使得反向传播穿过转移函数，迫使潜空间编码完整的动态信息
5. **压缩性能**：DVBF-LL在下界值上略优于DKF（798.56 vs 784.70），但下界差异不能反映生成采样质量的巨大差距

## 重要公式

**重参数化转移**：
$$z_{t+1} = f(z_t, u_t, \beta_t), \quad \beta_t \sim q_\phi(\beta_t|x_{1:T})$$

**DVBF变分下界**：
$$\mathcal{L}_{DVBF} = \mathbb{E}_{q_\phi}[\ln p_\theta(x_{1:T}|z_{1:T})] - \text{KL}(q_\phi(\beta_{1:T}|x_{1:T}, u_{1:T}) \| p(\beta_{1:T}))$$

**局部线性转移**：
$$z_{t+1} = A_t z_t + B_t u_t + C_t w_t, \quad A_t = \sum_{i=1}^{M}\alpha_t^{(i)}A^{(i)}$$

**退火目标**：
$$c_i = \max(1, 0.01 + i/T_A)$$

## 与我们项目的关联
DVBF对CVHI在动态系统辨识方面有直接启示：
- **完整信息潜空间**：CVHI推断的hidden species状态应包含恢复动态的完整信息（不仅是当前丰度，还有变化趋势/速度）。DVBF通过重参数化转移强制实现这一点的策略值得借鉴
- **状态空间模型框架**：生态系统可建模为状态空间模型，其中hidden species状态 h_t 遵循马尔可夫转移。DVBF的马尔可夫假设 p(h_{t+1}|h_t) 与生态系统的时间演化一致
- **控制输入**：DVBF支持控制输入 u_t，可类比为生态管理干预措施（如保护区设置、栖息地恢复等），使CVHI能建模人为干预对hidden species恢复的影响
- **长期预测**：DVBF的稳定长期预测能力对预测hidden species的未来恢复轨迹至关重要
- **与DVAE综述联系**：DVBF是Deep Kalman Filter (DKF) 的改进版本，属于DVAE家族

## 一句话总结
DVBF通过将状态转移重参数化为确定性函数并强制梯度通过时间传播，解决了此前时序VAE方法潜空间信息不完整的问题，实现了从原始数据中无监督学习完整信息状态空间模型。
